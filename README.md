FastButton

A local-first fasting tracker PWA.
- One big Start/End button (timestamps, so it works when the app is closed)
- Manual add for missed fasts
- Week/month/year totals
- Lifetime goal tracking (default: five thousand hours)
- Export/import backup

Deploy (Netlify)
- This is a static site (no build step).
- Publish directory: .

Install on Android
- Open the Netlify URL in Chrome.
- Use “Install app” or “Add to Home screen”.

Install on iOS
- Open the URL in Safari.
- Share → Add to Home Screen.

Notes
- All fasting data is stored locally in your browser storage.
- Use Export as a backup.
